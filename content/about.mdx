# About This Blog

A production-ready blog template built on Nextra v4, enhanced with modern features and zero configuration hassle.

## Why This Template?

The original Nextra blog theme is powerful, but getting started can be tricky. This template changes that. Clone it, customize your style, and deploy.

## Getting Started

```bash
git clone https://github.com/phucbm/nextra-blog-starter
cd nextra-blog-starter
pnpm install
pnpm dev
```

Visit `localhost:3000` and start writing in the `/content` directory.
